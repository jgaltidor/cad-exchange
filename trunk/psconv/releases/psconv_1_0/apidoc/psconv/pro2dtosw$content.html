Class Summary
trait Converter extends (Pro2D) => scala.Option[(Pro2D, SW2D)]
Represents a conversion function that converts a Pro/E subsection to a SolidWorks subsection. Let optconv be an instance of Converter. Let prosec be a Pro/E section. Suppose optconv finds a subsection of that it can convert and let prosubsec be this proe section. Let swsubsec be the SW section that prosubsec is converted to. optconv.apply(prosec) should return Some(prosubsec, swsubsec). If no subsection could be converted, then None is returned.
class DefaultConversion (prosec : Pro2D) extends OneToOneConv
class IllegalConversionException (msg : java.lang.String) extends java.lang.RuntimeException with AnyRef
trait LoggingConverter extends Converter with AnyRef
abstract class NonOptConverter extends Converter with AnyRef
Allows a Java class to easierly implement a subclass of Converter by having clients implement noOptApply instead of apply, which, does not return scala.None. Instead of returning None, noOptApply should return null
